## 项目概述

在线考试系统是基于Python的Django框架与Vue3的前后端分离架构开发而成，旨在为校园提供全面、稳定的考试管理解决方案。

## 技术栈

- **后端**: Python 3.x, Django, Node.js
- **数据库**: MySQL 8.0
- **前端**: Vue3

## 功能模块

- **学校信息管理**
- **班级信息管理**
- **学生信息管理**
- **教师信息管理**
- **考试科目管理**
- **校园考试管理**
